# SanthePriceIndex

SanthePriceIndex is a Kotlin-based Android application integrated with Firebase services. It is designed to track, manage, or index commodity/market prices ("Santhe" referring to traditional local markets). 

---

## 🚀 Features

* **100% Kotlin:** Built using modern Android development practices.
* **Firebase Integration:** Equipped with `google-services.json` for seamless backend support (Authentication, Firestore, or Realtime Database).
* **Gradle Kotlin DSL:** Modern and clean build configuration using `build.gradle.kts`.

---

## 🛠️ Tech Stack & Architecture

* **Language:** [Kotlin](https://kotlinlang.org/)
* **Build System:** Gradle Kotlin DSL (`.kts`)
* **Backend Services:** Firebase (Auth, Database, Analytics, etc.)
* **IDE:** Android Studio

---
